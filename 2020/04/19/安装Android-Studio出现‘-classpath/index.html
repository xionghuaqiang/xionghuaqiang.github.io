安装Android Studio出现‘'..classpath'

安装Android Stuido 出现“Could not resolve all artifacts for configuration ‘classpath’”

前几天不小心在清理垃圾的时候，把Android Sutdio删掉了，今天重新下的时候遇到点小问题，如标题，

在网上寻找方案后，找到一个能用的，如下

解决方案：
repositories {
       google()
       mavenCentral()  // add
       jcenter()

   }

添加如图代码即可，如一遍没有加载成功，多加载即便即可
